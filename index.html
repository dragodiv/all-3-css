Examples of Inline, Embedded and External CSS

INLINE CSS

The style attribute works in the same way as any other HTML attribute. We use style, followed by the equality sign (=), and then a quote where all of the style values will be stored using the standard CSS property-value pairs - "property: value;"

EMBEDDED CSS

The CSS syntax for embedded style sheets is exactly the same as other CSS code, apart from the fact that it is now wrapped within the <style> </style> tags. The <style> tag takes the "type" attribute that defines the type of style sheet being used (ie. text/CSS).

EXTERNAL CSS

An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage. Multiple webpages can use the same link to access the stylesheet.

"<link rel="stylesheet" href="style.css">"
